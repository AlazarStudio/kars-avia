- В настройках профиля, можно редактировать сам профиль
- Обновлен раздел "О компании" для отелей и авиакомпаний.

## v0.2 (2024-12-04)

- Обновлены фильтры по авиакомпаниям и аэропортам.
- Улучшена логика отмены заявок.
- Исправления в работе шахматки.
- Обновление номерного фонда, добавление описания и изображения.
- В разделе "Об Авиакомпании" добавлен подраздел, где отображаются номера

## v0.3 (2024-12-07)

- Добавлено хранение в локальном хранилище isAirline для отчетов, чтобы определить по авиакомпаниям или по гостиницам отчеты.
- Улучшена работа с выпадающими списками в DropdownList.
- Добавление DropdownList в половине сайта.

## v1.0 (2024-12-12)

- Выгрузили обновления на сайт.
- Обновлен интерфейс бокового меню, добавлена узкая версия меню.
- Добавлен новый мини-логотип для узкого меню.
- Улучшен поиск, теперь правильно работает с пагинацией, улучшена фильтрация заявок.
- Оптимизированы всплывающие подсказки.

## v1.1 (2024-12-18)

- Улучшена работа с заявками, при совпадении по датам появляется соответствующее сообщение и ссылка на заявку.
- Оптимизирован поиск по номерному фонду, правильно обрабатывается количество мест.

## v1.2 (2024-12-21)

- При изменении сотрудника авиакомпании, доработано изменение отдела

## v2.0 (2025-01-09)
