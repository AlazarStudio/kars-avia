- Обновлен раздел "О компании" для отелей и авиакомпаний.

### v0.2 (04.12.2024)
- Доработана фильтрация по авиакомпаниям и аэропортам.
- Улучшен механизм отмены заявок.
- Внесены исправления в работу модуля шахматки.

### v1.1 (18.12.2024)
